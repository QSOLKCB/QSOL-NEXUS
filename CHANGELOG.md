# Changelog

All notable changes to QSOL NEXUS are documented here. The project uses semantic versioning for the application and independent semantic versions for engines and artifact contracts.

## [1.0.0] — 2026-07-14

### Added

- Zero-install browser-native laboratory shell with fixed-order classic scripts and restrictive CSP.
- Canonical Strict, Replay Safe, and explicitly non-deterministic Creative execution modes.
- Canonical JSON, bundled SHA-256, versioned seeded PRNG, decimal parsing, cancellation and bounded scheduling.
- Shared matrix, eigensystem, statistics, graph, entropy, interpolation and optimization libraries.
- Offline stereo PCM generation, explicit PCM16 quantization, RIFF/WAV writer, deterministic store-only ZIP writer and signal fingerprinting.
- Frozen experiment recipe, observation contract, manifest, lineage, domain-separated hashes, no-self-hash pipeline, full bundle validation and recompute-not-trust replay.
- IndexedDB experiment history and local presets with visible session/memory fallback.
- Qutrit Field Laboratory with seeded initializers, three-state update rules, grid topologies, boundary conditions, measurements, field view, event stream and sonification.
- Tensor Resonance Laboratory with real symmetric 2×2/3×3 matrices, transformations, invariants, eigensystems, principal-axis view and quadrature/residual sonification.
- Galaxy Model Laboratory with strict SPARC-style adapter, explicit legacy-alias receipts, baryonic baseline, NFW, MOND simple interpolation, `uff_experimental_v1`, transparent grid/refinement fitting, comparison metrics, plots and sonification.
- Mandatory claim classes, model-status badges and identity-bearing claim boundaries.
- Browser-native known-answer, engine, audio, replay, tamper, cancellation and security tests.
- Complete architecture, determinism, artifact, model, security, attribution and archival documentation.

### Security

- No production network client, remote asset, analytics, service worker, dynamic script injection, `eval`, or `new Function`.
- ZIP import rejects unsafe paths, duplicates, unsupported compression, invalid CRCs and over-budget archives.

### Claim boundary

- Qutrit simulation is explicitly classical.
- Tensor quadrature is explicitly an audio observation, not physical self-duality.
- UFF is permanently labelled experimental for this engine version.
- Model comparison and sonification do not certify physical correctness.
